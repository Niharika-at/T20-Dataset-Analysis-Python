# T20 Data Analysis – 4th Semester Assignment

## Project Overview

This project analyzes an **International T20 Cricket dataset** using Python.
The analysis includes team performance, match results, venues, and scoreboard statistics.

The project is part of a **4th semester generic assignment** for data analysis using Python.

## Dataset

The dataset used in this project is **International T20 match data**.
It contains information about matches such as:

* Teams
* Venue
* Match winner
* Scores
* Player of the match
* City and country

File included:

* `International_T20_Data.csv`

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Project Files

* `main.py` – Runs the main program.
* `pair.py` – Shows relationships between match statistics.
* `scoreboard.py` – Displays match scoreboard analysis.
* `venue.py` – Analyzes matches played at different venues.
* `win.py` – Shows team win statistics.

## How to Run the Project

1. Install required libraries:

```
pip install pandas matplotlib seaborn
```

2. Run the main file:

```
python main.py
```
